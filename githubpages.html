Github Pages

Quando estamos fazendo sites e não temos dinheiro para assinar um plano de hospedagem ou é um projeto pessoal como um portifólio, podemos usar essa ferramento do Github, que é o Github Pages, ele serve para você hospedar seu site estático ou dinâmico de graça dentro do Github. Para realizar esse processo primeiro devemos conferir se os nossos arquivos do projeto já estão finalizados; agora entre no seu Github, clique em seu perfil e procure a aba repositórios. Depois disso crie um novo repositório clicando no botão new, que parece um livro e é esverdeado; Agora você foi levado até a página de criação de repositório, lá coloque um nome para esse repositório, uma descrição do que é, deixe ele publico para as pessoas poderem visualizá-lo, então clique em create repository. Até aqui criamos um repositório no github, agora vá para o git bash (que eu já mostrei como configurar e conectar ele no github) e digite git clone, agora copie o link do seu repositório lá do github, cole no git e adicione um ' .git ', e lembre de não dar enter no git clone antes de colar o link . Agora com o novo diretório criado entre nele usando o comando 'cd nome-do-repositório' no git. Agora mova os arquivos do seu projeto para essa pasta usando o comando 'git add .' , faça um commit com o comando ' git commit -m "Primeiro commit" ', agora é só enviar os arquivos usando o comando ' git push ' e agora já está lá os seus arquivos no repositório. No momento devemos entrar no github e acessar esse nosso repositório, agora dentro do repositório clique em 'settings' e procure por 'pages' , aqui vemos que existe algo chamado de build e deployment que tem o Source e deixe a opição de Deploy from a branch , e na categoria branch selecione a 'main' e do lado (root) e clique em save. Agora é só esperar um pouco e depois atualizar a página do github, quando tiver subido, você vera um link e do lado um botão que está escrito 'Visit site', clique no botão e então você tem um site no github. Se você quiser é só mandar o link do seu site para as pessoas que elas poderam acessesa-lo.
